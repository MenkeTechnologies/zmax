>_EXECUTIVE SUMMARY

All figures below are counted directly from the Rust source tree (Edition 2021, MSRV 1.90, license MPL-2.0), not estimated.

~SCRIPTING EMBEDS

zemacs embeds two scripting front-ends under zemacs-term/src/commands/scripting/ (420 LOC total). Per project design, the Emacs Lisp port lowers to a shared fusevm bytecode VM rather than carrying its own VM/JIT.
